# Roadmap

## v0.1 macOS Prototype

- Manual card logging
- Elixir estimation
- Card-cycle panels
- Basic macOS app bundle

## v0.2 macOS Capture

- ScreenCaptureKit window capture
- Live preview
- Suspicious event frame detection
- Event list

## v0.3 iOS Capture Scaffold

- Host iOS app
- Broadcast Upload Extension
- App Group shared transport
- Screen frame sampling
- Second-screen event output interface

## v0.4 Labeling

- Save event clips locally
- Local labeling UI
- Export dataset labels as JSON
- Negative-example labeling
- Private lab import workflow

## v0.5 Recognition

- Baseline recognition engine
- Candidate cards with confidence scores
- User confirmation before automatic logging
- Rule filters for summons, clone, buildings, and death spawns

## v0.6 Community Dataset

- Opt-in community dataset export
- Model versioning
- Recognition evaluation report
- More complete card catalog
